# Finance App

A full-stack personal finance app for tracking transactions, budgets, credit cards, and user profiles with a MongoDB-backed Express API.

Built with React, Vite, Node.js, Express, and MongoDB to provide a clean dashboard for managing everyday finances.

## Features

- User authentication
- Transaction tracking
- Budget management
- voice based transaction system
- Credit card management
- User profile support
- MongoDB database integration

## Tech Stack

- React
- Vite
- Node.js
- Express
- MongoDB
- Mongoose
